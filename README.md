# Mel Hair Profiles

## Install via VCC

```
https://mememelmelmel.github.io/hair-profiles/index.json
```

Add the URL above in VCC → Settings → Packages → Add Repository.

---

## Overview

Avatar compatibility profiles for hair assets sold at [めめめるめるめる](https://mememelmelmel.booth.pm/) on Booth.

Each profile is written in the format output by [Mel Profile Applier](https://github.com/mememelmelmel/profile-applier) and covers per-avatar adjustments for the supported hair assets.

This package itself does not include profile apply functionality. Install [Mel Profile Applier](https://github.com/mememelmelmel/profile-applier) if you want to apply profiles to your avatars.

**Primary use:** Centralized management of avatar support profiles by the author. Users can also check avatar compatibility status and use the profiles as a reference for adjustments in their own environments.

---

## 概要

Booth ショップ「[めめめるめるめる](https://mememelmelmel.booth.pm/)」で販売している髪型アセットの、各アバター向け対応プロファイルです。

各プロファイルは [Mel Profile Applier](https://github.com/mememelmelmel/profile-applier) の出力フォーマットで記載されており、対応アバターごとの調整値が含まれます。

このパッケージ自体にプロファイルの適用機能はありません。プロファイルをアバターに適用したい場合は [Mel Profile Applier](https://github.com/mememelmelmel/profile-applier) を別途インストールしてください。

**主な用途:** 製作者による対応アバタープロファイルの一元管理。ユーザーは対応状況の確認や、個人環境での調整の参考としても活用できます。
